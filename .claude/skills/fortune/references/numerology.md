# Numerology Reference — Life Path Numbers

## How to Calculate
Sum all digits of the full birth date (DDMMYYYY) repeatedly until a single digit (or 11, 22, 33).

Example: 05/03/1990 → 0+5+0+3+1+9+9+0 = 27 → 2+7 = **9**
